NAME

Mojolicious::Plugin::Jam::Form - HTML form object

SYNOPSIS

 my $form = Mojolicious::Plugin::Jam::Form->new;
 
 # Add and configure fields
 $form->field;
 
 # Generate html
 print $form->html;
 print "$form";
 
 # Set values from param
 $form->values($self);
 
 # Validate form
 if ($form->valid) {
   # do something
 } else {
   # do something else
 }

DESCRIPTION

Mojolicious::Plugin::Jam::Form provides an object-based representation of an HTML form for use with the 'form' command in Mojolicious::Plugin::Jam.

It exhibits the following features:

  • Modern Perl-ish API allows method chaining

PROPERTIES

id

Get/set the form id

action

Get/set the form action

method

Get/set the form method

fields

Get/set/append to form fields arrayref

METHODS

get

Set form method to 'GET'. Returns Mojolicious::Plugin::Jam::Form object.

post

Set form method to 'POST'

field

Create and configure a Mojolicious::Plugin::Jam::Form::Field object and add to the form object. Returns the Mojolicious::Plugin::Jam::Form::Field object.

html

Generate and return the HTML for the form. This method is called when the object is treated as a string via overloading.

values

Set values from the controller prior to performing validation.

valid

Call the valid method on each Mojolicious::Plugin::Jam::Form::Field object in the form. Returns 0 if any fails, 1 if all pass.

BUILT-IN FIELDS

Built-in fields are extremely common fields which should help enable Mojolicious developers to rapidly assemble forms. In time, the number of built-in fields should grow.

text

Creates and adds a text input field to the form

username

Creates and adds a username input field to the form

password

Creates and adds a password input field to the form

submit

Creates and adds a submit button to the form

reset

Create and adds a reset button to the form
